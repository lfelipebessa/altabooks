Pipeline de Catalogação Automática

Estes mockups representam o desenho visual e de fluxo do novo módulo Agente Verde dentro da plataforma da Alta Books. São telas estáticas — não há lógica, banco de dados ou IA por trás. Servem para validar UX e fluxo antes de qualquer linha de código de aplicação.

Telas

📋

1. Listagem de Lotes

Tela inicial do módulo. Mostra todos os prelos já enviados, com status agregado (quantos prontos, pendentes, com falha). Botão pra fazer upload de um novo prelo.

→
✏️

2. Revisão de Lote

Tela central — onde Cristiane / Anderson revisam cada ISBN. Layout mestre-detalhe: lista de itens à esquerda, editor com tabs Book Info / Metagráfica à direita. Aprovação item-a-item ou em massa.

→
⬆️

3. Upload de Novo Prelo

Modal que abre a partir da listagem. Drag-and-drop da planilha .xlsx do prelo. Preview dos ISBNs detectados antes de confirmar o disparo do processamento.

→

Fluxo de uma catalogação

Cristiane envia o prelo do mês ──→ [tela 3: upload]
                                          ↓
                          Sistema lê os ISBNs (ex: 50 livros)
                                          ↓
                          Pra cada ISBN, em paralelo (n8n):
                          ├─ Busca pasta no Drive corporativo
                          ├─ Confere 3 arquivos (capa, miolo, técnicos)
                          ├─ Se falta arquivo → "pending_files", pula
                          ├─ Extrai texto/imagem dos PDFs
                          ├─ Lê planilha técnica do Anderson
                          ├─ Chama Claude com a base de conhecimento
                          │   (BISAC, marcas, taxonomia, templates)
                          ├─ Calcula lombada, peso, dimensões
                          └─ Preenche 2 cadastros (físico + digital)
                                          ↓
                              Lote aparece na listagem
                              com contagem agregada
                                          ↓
                          Cristiane abre → [tela 2: revisão]
                          ├─ Vê 42 prontos, 6 pendentes, 2 falhas
                          ├─ Clica item-a-item, ajusta, aprova
                          └─ "Aprovar todos selecionados"
                                          ↓
                          Exporta Book Info.xlsx + Metagráfica.xlsx
                          → entra no fluxo da editora
